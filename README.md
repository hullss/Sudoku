# Sudoku (студентський проєкт)

Невелика desktop-гра **Sudoku** на Java з графічним інтерфейсом на базі `jpaz2`.

## Можливості
- Генерація нової судоку-дошки.
- Вибір складності через кількість порожніх клітин:
  - `40` (легше)
  - `50` (середньо)
  - `60` (складніше)
- Перевірка конфліктів у рядках, стовпцях і блоках `3x3` (помилки підсвічуються червоним).
- Скидання поточної гри до початкового стану.
- Збереження та завантаження стартової дошки у файл `sudoku_start.csv`.

## Технології
- Java `8`
- Maven
- Бібліотека `sk.upjs:jpaz2:1.1.1`

## Запуск

### 1) Запуск через Maven
```bash
mvn clean package
mvn exec:java -Dexec.mainClass=sk.upjs.paz.Launcher
```

### 2) Запуск з готового JAR
Після збірки формується fat-jar:
```bash
java -jar target/Sudoku.jar
```

## Керування
- Клік по клітинці: вибір клітинки.
- Клавіші `1..9`: ввести число.
- `0`, `Backspace`, `Delete`: очистити клітинку.
- Стрілки: переміщення вибору по полю.
- `R`: reset (скинути до початкового стану).
- `N`: швидко згенерувати нову гру.
- Кнопки праворуч у вікні:
  - вибір складності (`40/50/60`)
  - `New game`
  - `Reset`
  - `Load` / `Save` (файл `sudoku_start.csv`)

## Формат файлу збереження
Файл `sudoku_start.csv` має містити рівно `9` рядків по `9` чисел (`0..9`) у кожному.
- `0` означає порожню клітинку.
- Розділювачі: кома або пробіли.

Приклад:
```csv
0,6,7,0,9,0,0,0,3
0,0,0,0,0,8,0,0,0
2,0,0,4,0,0,7,0,0
0,0,0,0,0,4,0,0,5
0,0,0,0,5,0,9,0,0
0,0,0,9,0,0,3,0,0
0,0,0,5,4,0,0,0,0
0,0,0,0,0,9,2,0,0
0,0,0,2,0,0,5,0,4
```

## Структура проєкту
- `src/main/java/sk/upjs/paz/Launcher.java` - точка входу.
- `src/main/java/sk/upjs/paz/SudokuPane.java` - UI та обробка подій.
- `src/main/java/sk/upjs/paz/SudokuBoard.java` - логіка дошки.
- `src/main/java/sk/upjs/paz/SudokuGenerator.java` - генерація нових пазлів.
- `src/main/java/sk/upjs/paz/GameState.java` - збереження/завантаження стану.
- `src/main/resources/best.png` - ресурс зображення для UI.
